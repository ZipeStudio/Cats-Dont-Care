**Cats Don't Care** — Lets cats find cozy spots to sit and sleep around your world.

### Showcase
![showcase](https://github.com/ZipeStudio/cats-dont-care/blob/master/img/showcase.png?raw=true)

!["Compatibility" Title](https://github.com/ZipeStudio/Vault/blob/main/design/mods/main/ZScompatible.png?raw=true)

Compatibility blocks:
- all wool carpets
- moss carpet
- hay block
- furnace
- barrel

!["Configurable" Title](https://github.com/ZipeStudio/Vault/blob/main/design/mods/main/ZSconfigurable.png?raw=true)

You can add your own blocks for cats to sit and sleep on using a datapack.<br>
Create the following file:
`data/cats_dont_care/tags/block/cozy_spots.json`

Example:
<pre style="border-left:3px solid #4CAF50; padding:10px;"><code>{
"replace": false,
"values": [
"minecraft:crafting_table",
"mod_id:block_id"
]
}
</code></pre>

Any block added to this tag will be considered a cozy spot for cats.
